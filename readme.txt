基于javascript开发的客户端应用框架，使我们可以更加快捷，简单的开发web应用。
适用于 CRUD 应用、前后端分离的开房方式、或者 SPA 单页面网站的开发。
不适合做一些很酷炫的特效、游戏、图形界面。
